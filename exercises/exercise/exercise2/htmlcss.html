NOTE
We recommend saving an exact version of
DevExtreme to avoid unexpected updates
because DevExtreme does not use Semantic
Versioning. In our versioning system, the first
and middle numbers indicate a major release
which may contain behavior changes.

You need additional npm packages to use
DevExtreme with Angular, Vue or React. See
the following instructions: Angular, Vue, React.
